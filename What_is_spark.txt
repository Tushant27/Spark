what is Apache Spark?
So the definition is Apache Spark is a unified computing engine and set of libraries for parallel data processing on computer clusters.

What is unified?
Unified, instead of having separate disjoint pools for different tasks like one of the batch jobs, another for streaming, and so on, Spark integrates them all.

what is computing engine?
Core part that performs calculation or data operation in Spark, it is the engine that distributes tasks across cluster, processing data in parallel and handles all the lifting.
